# Market Dashboard

A live investor market dashboard hosted on GitHub Pages, auto-updated daily by a GitHub Action.

**Live site:** `https://YOUR_USERNAME.github.io/market-dashboard`

---

## What's included

| File | Purpose |
|---|---|
| `index.html` | The dashboard — works standalone with seeded data |
| `data.json` | Auto-generated by GitHub Actions with live market data |
| `scripts/fetch_data.py` | Fetches data via `yfinance` (no API key needed) |
| `.github/workflows/update-data.yml` | Runs the script at 6 PM ET weekdays |

---

## Deploy to GitHub Pages in 5 steps

### 1. Create the repo

```bash
cd market-dashboard
git init
git add .
git commit -m "initial commit"
```

Go to [github.com/new](https://github.com/new) and create a repo named `market-dashboard`.

```bash
git remote add origin https://github.com/YOUR_USERNAME/market-dashboard.git
git branch -M main
git push -u origin main
```

### 2. Enable GitHub Pages

1. Go to your repo → **Settings** → **Pages**
2. Under **Source**, select **Deploy from a branch**
3. Branch: `main`, folder: `/ (root)`
4. Click **Save**

Your site will be live at `https://YOUR_USERNAME.github.io/market-dashboard` within ~60 seconds.

### 3. Generate initial data.json

Run the fetch script locally (requires Python + yfinance):

```bash
pip install yfinance requests
python scripts/fetch_data.py
git add data.json
git commit -m "chore: initial data"
git push
```

Or trigger the Action manually: **Actions tab** → `Update market data` → **Run workflow**.

### 4. Verify the Action works

The workflow runs automatically every weekday at 6 PM ET. To check:
- **Actions tab** in your repo
- Click `Update market data`
- You should see green checkmarks

### 5. Update `index.html` with your GitHub URL

Replace `YOUR_USERNAME` in the footer link in `index.html`.

## Customizing your portfolio tickers

The dashboard loads `AAPL, NVDA, MSFT, JPM, XOM` by default. To change the default tickers fetched by the Action, edit `scripts/fetch_data.py`:

```python
DEFAULT_TICKERS = ["AAPL", "NVDA", "MSFT", "JPM", "XOM"]
# Change to whatever you want, e.g.:
DEFAULT_TICKERS = ["TSLA", "META", "AMZN", "BRK-B", "GLD"]
```

You can also add tickers interactively on the website — they're saved to your browser's `localStorage`.

## Updating the data manually

```bash
python scripts/fetch_data.py
git add data.json && git commit -m "chore: refresh data" && git push
```

Or click **Run workflow** in the Actions tab.

## Data sources

- Market data: [yfinance](https://github.com/ranaroussi/yfinance) (Yahoo Finance) — free, no API key
- Macro data (CPI, unemployment, Fed rate): manually maintained in the script; update `macro` list in `fetch_data.py` when new figures are released
- Sector ETFs: XLK, XLF, XLV, XLE, XLU, XLY, XLP, XLI, XLB, XLRE, XLC

## Troubleshooting

**Page shows "—" for all values**
→ `data.json` hasn't been generated yet. Run `python scripts/fetch_data.py` locally and push, or trigger the Action manually.

**Action fails with permissions error**
→ Go to **Settings → Actions → General → Workflow permissions** and set to "Read and write permissions".

**yfinance rate-limited**
→ Wait a few minutes and re-run. The seeded data in `index.html` will still display if `data.json` is unavailable.
